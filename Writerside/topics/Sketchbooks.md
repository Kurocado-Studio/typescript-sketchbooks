# Sketchbooks

The Sketchbooks workspace contains exploratory front-end work developed across React and Vue.

It is used to test UI ideas, evaluate SystemHaus token behavior, prototype motion patterns, and
document TypeScript-first architectural approaches.

---

## Sections

### Daily UI

Interface and component experiments, including Storybooks, layout studies, interaction tests, and
token-driven variations.

- **[View Daily UI sketchbook](Daily-UI-React.md)**

### Motion

Motion behavior experiments covering transitions, timing, easing, gesture handling, and reusable
motion primitives.

- **[View Motion sketchbook](Motion.md)**

### TS Patterns

Examples focusing on TypeScript modeling, render patterns, state management variants, and
composition strategies for React and Vue.

- **[View TS Patterns sketchbook](Patterns.md)**
